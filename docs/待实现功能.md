当前状态：已完成首版。`FinancialEvaluator` v5 对所有证券执行线上硬门禁并把来源可用性降为告警；`FinancialEvaluationRunner` 使用 12 个股票/ETF 冻结正例、8 个规则负例和 24 个检索问题，输出 JSON/Markdown，执行 Recall@K、Precision@K、MRR、NDCG@K、数值误差和基线回归。PR GitHub Actions 已接入确定性强门禁，Judge/真实数据工作流保持非阻断。
