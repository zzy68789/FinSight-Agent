# 目前已经实现的功能

更新时间：2026-07-06

## A股/ETF研究报告 MVP

- 新增 `POST /api/stock-reports` SSE 入口，支持 `ticker`、`thread_id`、`report_period`、`search_mode`。
- 股票/ETF 代码解析支持普通 A 股 `6xxxxx -> .SH`、`0xxxxx / 2xxxxx / 3xxxxx -> .SZ`，以及常见 ETF `5xxxxx -> .SH`、`15xxxx / 16xxxx / 18xxxx -> .SZ`。
- 不支持的代码会通过 SSE `error` 事件返回明确业务原因，不再在前端泛化为“网络异常”；当前仍不支持 B 股、债券或北交所标的。
- 新增金融数据模型：`StockSubject`、`FinancialEvidenceItem`、`FinancialSnapshot`、`FinancialMetricResult`。
- 新增 `FinancialDataProvider` 扩展点，已实现 `UploadedReportProvider`、`PublicMarketDataProvider` 和 `TushareMarketDataProvider`。
- 新增 `AShareCompanyDirectory` 和 `AShareMasterDataProvider`，为 10 个默认评测样例股票补齐公司名称、行业和 `LOCAL_CONTEXT` 证据。
- 新增 TuShare Pro 授权数据源配置，支持 `DRAI_TUSHARE_ENABLED`、`DRAI_TUSHARE_API_KEY`、`DRAI_TUSHARE_BASE_URL`、`DRAI_TUSHARE_TIMEOUT`；未配置 token 时不调用外部接口。
- `TushareMarketDataProvider` 已能把普通股票的 TuShare `income`、`balancesheet`、`cashflow`、`daily_basic` 响应映射为营业收入、营业成本、归母净利润、资产负债、经营现金流、市盈率、市净率和总市值证据；ETF 标的优先走 `fund_daily` 并映射收盘价、涨跌幅和成交额证据。
- `FinancialSnapshotBuilder` 已支持 provider fan-out 并行采集，并为每个数据源记录状态、耗时、证据数量和失败信息。
- 通用研究任务、股票报告任务和金融 provider fan-out 已改为 Spring 管理的有界执行器，线程数可通过 `DRAI_WORKFLOW_THREADS` 和 `DRAI_FINANCIAL_PROVIDER_THREADS` 配置。
- 公开数据失败时不会阻断流程，会写入 `DATA_MISSING` 证据。
- 新增 `FinancialMetricEngine`，普通股票用 Java `BigDecimal` 计算营收同比、毛利率、净利率、ROE、资产负债率、经营现金流 / 净利润；ETF 不套用公司财务指标，只输出 ETF 收盘价、涨跌幅和成交额等行情指标。
- 缺少分子或分母时返回 `MISSING_INPUT`，报告中显示“数据缺失”。
- 新增 `FinancialRiskScoringService`，按基本面 30%、技术面 25%、情绪面 20%、消息面 15%、市场环境 10% 输出五维风险评分、风险等级和缺失证据 warning。
- 新增 `FinancialComplianceReviewer`，检查免责声明、保证收益、内幕信息和引用审查失败，并输出结构化 `issues` 与合规分数。
- 新增 `FinancialEvaluationService`，加载 `financial-eval-set.json` 并输出 claim/citation/numeric/keypoint 自动评测结果。
- 新增独立金融工作流：`StockResolve -> DataSnapshot -> MetricEngine -> RiskAssessment -> EvidenceCollect -> InvestmentWriter -> CitationReviewer + ComplianceReviewer -> Evaluation`。
- 普通股票报告和 ETF 报告均固定八章节，并在底部输出“引用与数据快照”和指标公式；ETF 报告使用基金概况、跟踪标的、行情、流动性、净值/持仓缺口等基金口径章节。
- 新增 `CitationReviewer`，检查有效证据数量、报告期混用、指标证据引用和指标值展示；最终通过条件为引用审查与合规审查同时 PASS。
- 报告统一标注“仅作研究辅助，不构成投资建议”。
